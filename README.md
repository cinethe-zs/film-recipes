# Film Recipes

A single-page reference tool for emulating analog film stocks on vintage digital cameras using in-camera JPEG settings — no Lightroom presets, no RAW editing.

**192 film stocks** across color negative, slide, B&W negative, and experimental emulsions — each with tailored white balance, exposure, and picture style settings for 6 cameras.

## Cameras supported

- Sony NEX-3N
- Panasonic Lumix LX3
- Panasonic Lumix GF1
- Panasonic Lumix GM5
- Panasonic Lumix GX80
- Olympus OM-D E-M10 Mark II

## Features

- Filter recipes by film type (color, B&W, slide, cross-process, experimental)
- Filter by push/pull compensation
- Compare two recipes side by side
- Random recipe picker
- Print-friendly view

## Usage

Open `index.html` in any browser — no build step, no dependencies, no server required.
